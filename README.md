# Django Blog Application

Современный и лаконичный веб-сайт блога, построенный на Django и оформленный с помощью минималистичного CSS-фреймворка Pico.css.

## Основные возможности

* **Управление контентом**: Просмотр списка постов с предпросмотром картинок и краткой информацией.
* **Детальный просмотр**: Отдельная страница для каждого поста со всем содержимым.
* **Система комментариев**: Авторизованные пользователи могут оставлять комментарии к публикациям.
* **Аутентификация**: Вход/выход из системы для пользователей (`django.contrib.auth`).
* **Пагинация**: Динамическая разбивка списка публикаций по страницам (по 3 поста на страницу).
* **Адаптивный UI**: Чистый интерфейс без громоздких CSS-библиотек на базе Pico.css.

## Технологический стек

* **Backend**: Python 3.x, Django 5.x
* **Frontend**: HTML5, Pico.css (CDN)
* **Database**: SQLite (по умолчанию)
